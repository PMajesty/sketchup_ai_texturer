# AI Texture Generator (Sketchup Plugin)

This is a toy project I built to learn the SketchUp Ruby API.

For now, it doesn't use AI - it simply picks a random texture from a local asset library and applies it to selected faces. It was a playground for understanding `Sketchup::Model`, `Materials`, and extension registration.

## Quick Start

1.  Copy the `sketchup_ai_texturer.rb` file and the `sketchup_ai_texturer` folder into your SketchUp Plugins directory.
2.  Restart SketchUp.
3.  Select one or more faces in your model.
4.  Navigate to **Plugins -> Generate AI Texture**.
